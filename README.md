# ai_chatbot_flask_react
AI Chatbot (React + Flask)

 Features
- Chatbot UI using React (Vite)
- Flask backend API
- Offline AI using Ollama (Mistral/LLaMA) for testing only further improve

🛠 Tech Stack

- Frontend: React (Vite)
- Backend: Flask (Python)
- AI Model: Ollama

⚙️ Setup Instructions

1. Start Ollama
   
ollama run mistral

2. Run Backend
python app.py

3. Run Frontend
npm install
npm run dev
